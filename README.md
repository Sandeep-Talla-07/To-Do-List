# To-Do List Web Application

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript. This project helps users manage their daily tasks efficiently by allowing them to add, complete, and remove tasks.

# Features
  - Add new tasks
  - Mark tasks as completed
  - Delete tasks
  - Automatic data persistence using Local Storage
  - Responsive and user-friendly interface
  - Modern gradient-based UI design

# Technologies Used
  - HTML5
  - CSS3
  - JavaScript (ES6)
  - Browser Local Storage

# How It Works
  - Enter a task in the input field.
  - Click the "Add" button to add the task.
  - Click on a task to mark it as completed.
  - Click the × icon to delete a task.
  - Tasks are automatically saved in Local Storage and remain available even after refreshing the page.
